# Digital Kamtibmas

Sistem Informasi Digital Kamtibmas Polrestabes Palembang

## Deskripsi

Modul Odoo 19 untuk pengelolaan data kamtibmas digital yang mencakup:
- **Satlantas**: Antrian layanan, e-Form laka, display antrian
- **Satnarkoba**: Konseling online, rehabilitasi narkoba
- **Satreskrim**: Manajemen barang bukti
- **Sabhara**: Tracking patroli

## Fitur Utama

- Dashboard interaktif dengan visualisasi data
- Peta interaktif dengan Leaflet.js
- Chart statistik dengan ECharts
- Sistem antrian digital dengan display real-time
- Workflow management untuk setiap satuan

## Teknologi

- Odoo 19.0
- Python 3.12
- PostgreSQL
- Leaflet.js
- ECharts
- OWL Framework

## Instalasi

1. Copy folder `digital_kamtibmas` ke direktori `addons` Odoo
2. Update apps list di Odoo
3. Install module "Digital Kamtibmas"
